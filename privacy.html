SummarizerX Privacy Policy / 隐私政策

Last updated: 2025-09-06

Data We Collect / 收集的数据

  • Page content / 页面内容: text you ask the extension to summarize or translate.
  • API key & settings / API 密钥与设置: saved in chrome.storage.sync on your device.

How We Use Data / 数据用途

  • Page content is sent to the selected API service (OpenAI, DeepSeek, etc.) to generate summaries or translations.
  • API keys and settings are used only for those requests and never shared elsewhere.

Storage & Transmission / 存储与传输

  • Page content is transmitted directly from your browser to the chosen API and is not stored by SummarizerX.
  • Settings, including API keys, remain locally in your browser's sync storage and are never uploaded by the extension.
  • Enabling Local Mode in settings prevents any page content from leaving your device; remote AI features will be disabled.

For questions or concerns, please open an issue on GitHub.
